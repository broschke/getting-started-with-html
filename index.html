Major pieces of the web:

Browsers - a program that runs on your computer to display files found on the web
The internet - the world's largest computer network
HTTP - protocol to connect browser to server
Servers - computers the host the files that make up the internet

HTML Markup
Consists of Tags and Content. Tags tell the computer to how display the contents. Together this is called an Element. Format is contents or contents

Examples of tags the b tag makes content bold
the em tag makes content italicised
the anchor tag, a, make links in a webpage udacity where href is an attribute and www.udacity.com is the value
the image tag format is as follows:
"<"img src="url" alt="text">"
where url is where the image is downloaded from and the alt or alternate is the text that is displayed when the image fails to load
image tags have no closing tag because it has no content. Also called void tag

Whitespace tags
the br tag creates a line break
This moves text after the tag to a new line. the p tag renders content in one paragraph. since it has content, it requires a closing tag br tag is a void tag
HTML code does not need to follow the action of the whitespace tags
Inline vs Block - the br tag is an inline element, simply text.
the p tag is a block element, an invisible block is created around the content of a p tag

span and div elements - simply contain their content but have the ability to edit how the content is laid out with CSS
span is inline while div is block. both require closing tags
